\doxysubsection{Detailed Description}
Represents a path with both driving and walking segments, along with associated metadata. 

This structure is used to store information about a route that includes\+:
\begin{DoxyItemize}
\item A driving path with its duration.
\item A parking node where the driving ends.
\item A walking path with its duration.
\item The total time for the entire route.
\item A message providing additional details about the path. 
\end{DoxyItemize}
