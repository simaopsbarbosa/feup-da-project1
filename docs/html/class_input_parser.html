std::vector< int > InputParser::parseNodes ( const std::string & input)

Parses a string of integers into a vector of nodes.

The input string should contain integers separated by spaces. The algorithm iterates through the input string once, making it's complexity O(n), where n is the number of integers.

Parameters
inputThe input string containing node data.
Returns
A vector of integers representing the nodes.

std::vector< std::pair< int, int > > InputParser::parseSegments ( const std::string & input)

Parses a string of segments into a vector of pairs of integers.

The input string should contain segments in the format "(x,y)", separated by spaces or other delimiters. The algorithm processes each segment in the input string, making its complexity O(m), where m is the number of segments.

Parameters
inputThe input string containing segment data.
Returns
A vector of pairs of integers representing the segments.
